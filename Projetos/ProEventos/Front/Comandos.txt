Instalar Angular CLI: npm install -g @angular/cli

Habilitar powershell: Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned

Criar aplicação: ng new my-app

Executar aplicação: ng serve --open, npm start

Altearar politica de scripts(Administrador): set-ExecutionPolicy RemoteSigned

Criar componentes: ng g c nome

Criar pipe: ng g p nome

Diretivas: *ngFor

Interpolação: {{ e.eventoId }}

ngOnInit: Método chamado antes de interpretar o html

Instalar font awsome: npm install --save @fortawesome/fontawesome-free

Instalar Angular Bootstrap: npm install ngx-bootstrap --save

Instalar toastr: npm install ngx-toastr --save

Instalar Spinner: $ npm install ngx-spinner --save

    @import "../node_modules/@fortawesome/fontawesome-free/css/all.min.css";
